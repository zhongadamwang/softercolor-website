## HOW

### Implementation Approach
- **Stack:** PHP (lightweight, no heavy framework required for static marketing site), MySQL available if needed for form submission logging
- **Frontend:** HTML5 + CSS3 + minimal JavaScript — no SPA framework needed
- **Deployment:** SFTP to HostGator cPanel
- **Content:** Hardcoded (infrequent updates) — no CMS
- **Contact form:** PHP mail handler or SMTP library (PHPMailer recommended for reliability)
- **SEO:** Static meta tags per page, sitemap.xml, robots.txt, schema.org JSON-LD

### Risk Mitigation
| Risk | Mitigation |
|------|-----------|
| Email delivery unreliability with PHP mail() | Use PHPMailer + SMTP |
| HostGator PHP version constraints | Confirm PHP 8.x support before build |
| SSL not auto-provisioned | Enable Let's Encrypt via cPanel |
| No analytics baseline | Set up GA/Plausible before launch |
